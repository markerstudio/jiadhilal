If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand.

## Quick map
- `readme.md` — full brand guide: colors, content fundamentals, visual foundations, iconography, index.
- `styles.css` — global CSS entry (link this); `@import`s everything in `tokens/`.
- `tokens/` — `colors.css`, `typography.css`, `spacing.css`, `effects.css`, `fonts.css`.
- `assets/` — logo lockup, logo mark (white/purple/black/gradient), app icon.
- `components/` — React UI primitives (Button, Input, Select, Checkbox, Switch, IconButton, Badge, Tag, ProgressBar, ProgressRing, StatTile, Avatar, Card, Tabs).
- `guidelines/` — foundation specimen cards.
- `ui_kits/coaching-app/` and `ui_kits/marketing-site/` — full product recreations.

## Brand in one breath
Electric violet (`#6200ff`), char-black, neon green/red sparks. Heavy oblique UPPERCASE
display (Archivo), Archivo black-italic for expressive accents, mono numerals for data. Voice: a
coach in your corner — direct, disciplined, no emoji, results-first. Sharp 45° geometry,
purple glow, snappy spring motion.
